## Overview

`FilaChipsFiltro` provides a horizontally scrollable row of Material 3 FilterChips for sorting the animal list. It supports three filter options and integrates with a date picker for date-based filtering.

## Component Signature

```kotlin
@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun FilaChipsFiltro(
    filtroActual: OpcionFiltro,
    fechaSeleccionada: Long?,
    alSeleccionarFiltro: (OpcionFiltro) -> Unit,
    alSolicitarCalendario: () -> Unit,
    modifier: Modifier = Modifier
)
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `filtroActual` | `OpcionFiltro` | Currently selected filter option |
| `fechaSeleccionada` | `Long?` | Selected date timestamp in milliseconds (UTC), used only for `POR_FECHA` filter |
| `alSeleccionarFiltro` | `(OpcionFiltro) -> Unit` | Callback invoked when a filter option is selected |
| `alSolicitarCalendario` | `() -> Unit` | Callback to open the date picker dialog |
| `modifier` | `Modifier` | External layout modifier (optional) |

## Filter Options

The component supports three filter types defined in `OpcionFiltro` enum:

### RECIENTES
- **Icon**: `Icons.Outlined.TrendingUp`
- **Label**: "Recientes"
- **Behavior**: Shows most recently registered animals first

### POR_FECHA
- **Icon**: `Icons.Outlined.CalendarMonth`
- **Label**: "Por fecha" or formatted date when selected
- **Behavior**: Opens date picker, filters to animals registered on selected date
- **Special Feature**: Label changes to show selected date (e.g., "15 Mar 2026")

### ANTIGUOS
- **Icon**: `Icons.Outlined.TrendingDown`
- **Label**: "Antiguos"
- **Behavior**: Shows oldest registered animals first

### Dynamic Date Label

When the "Por fecha" filter is selected and a date is chosen, the chip label updates to display the formatted date:

```kotlin
val etiqueta = if (opcion == OpcionFiltro.POR_FECHA && estaSeleccionado && fechaSeleccionada != null) {
    val cal = Calendar.getInstance(TimeZone.getTimeZone("UTC")).apply { 
        timeInMillis = fechaSeleccionada 
    }
    DateFormat.format("dd MMM yyyy", cal).toString()
} else {
    opcion.etiqueta
}
```

## Usage Example

```kotlin
import androidx.compose.runtime.*
import com.example.huellitas.model.OpcionFiltro
import com.example.huellitas.ui.components.FilaChipsFiltro

@Composable
fun AnimalListScreen() {
    var filtroActual by remember { mutableStateOf(OpcionFiltro.RECIENTES) }
    var fechaSeleccionada by remember { mutableStateOf<Long?>(null) }
    var mostrarDatePicker by remember { mutableStateOf(false) }

    Column {
        FilaChipsFiltro(
            filtroActual = filtroActual,
            fechaSeleccionada = fechaSeleccionada,
            alSeleccionarFiltro = { filtro ->
                filtroActual = filtro
                // Apply filter logic
            },
            alSolicitarCalendario = {
                mostrarDatePicker = true
            }
        )

        if (mostrarDatePicker) {
            DatePickerDialog(
                onDateSelected = { timestamp ->
                    fechaSeleccionada = timestamp
                    filtroActual = OpcionFiltro.POR_FECHA
                    mostrarDatePicker = false
                },
                onDismiss = { mostrarDatePicker = false }
            )
        }

        // Animal list content
        LazyColumn {
            items(filteredAnimals) { animal ->
                TarjetaAnimal(animal = animal)
            }
        }
    }
}
```

The component is stateless - all state is managed by the parent:

```kotlin

    fechaSeleccionada = fechaSeleccionada,
    alSeleccionarFiltro = { filtroActual = it },
```

## Date Format

Dates are formatted using the pattern `"dd MMM yyyy"` which produces output like:
- "15 Mar 2026"
- "01 Jan 2025"
- "30 Dec 2024"

The format uses the device's locale for month abbreviations.
