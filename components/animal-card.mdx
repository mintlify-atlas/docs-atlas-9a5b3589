## Overview

`TarjetaAnimal` is the primary card component for displaying animal information throughout the app. It features an asynchronous image loader with caching, a type badge, and organized information display with icons.

## Component Signature

```kotlin
@Composable
fun TarjetaAnimal(
    animal: Animal,
    modifier: Modifier = Modifier
)
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `animal` | `Animal` | The animal data to display including name, type, breed, description, location, contact, and image URL |
| `modifier` | `Modifier` | External layout modifier (optional) |

## Features

### Image Handling

- **Asynchronous Loading**: Uses Coil library for efficient image loading from URLs
- **Caching**: Images are cached in both memory and disk for performance
  - Memory cache policy: `ENABLED`
  - Disk cache policy: `ENABLED`
  - Image size: 600x400px
- **Crossfade Animation**: Smooth transition when images load
- **Placeholder**: Shows a paw icon when no image is available
- **Zoom Dialog**: Tap image to open full-screen zoom view

### Type Badge

Displays a colored badge in the top-right corner indicating animal type:

- **Dog** (`PERRO`): Blue badge using `DogChip` color
- **Cat** (`GATO`): Purple badge using `CatChip` color  
- **Other** (`OTRO`): Gradient purple using `GradientStart` color

Source: `/home/daytona/workspace/source/app/src/main/java/com/example/huellitas/ui/components/AnimalCard.kt:140-160`

### Information Display

The card organizes information with icon-text rows:

1. **Name**: Large title with bold font weight
2. **Breed**: Medium body text (shown if not empty)
3. **Description**: Truncated to 2 lines maximum
4. **Location**: Icon with location text
5. **Contact**: Icon with contact information
6. **Registration Date**: Formatted as "dd MMM yyyy"

## Usage Example

```kotlin
import com.example.huellitas.model.Animal
import com.example.huellitas.model.TipoAnimal
import com.example.huellitas.ui.components.TarjetaAnimal
import java.util.Date

@Composable
fun AnimalListScreen() {
    LazyColumn {
        items(animals) { animal ->
            TarjetaAnimal(
                animal = animal,
                modifier = Modifier.padding(16.dp)
            )
        }
    }
}
```

## Interactive Features

### Image Zoom

When the user taps on an animal image, the card opens a full-screen zoom dialog:

```kotlin
var mostrarZoom by remember { mutableStateOf(false) }

if (mostrarZoom && animal.imagenUrl != null) {
    DialogoZoomImagen(
        imagenUrl = animal.imagenUrl,
        descripcion = "Foto de ${animal.nombre}",
        alCerrar = { mostrarZoom = false }
    )
}
```
