## Overview

Huellitas uses a custom Material 3 theme with a warm purple palette inspired by animal rescue and compassion. The theme maintains consistent branding across all devices by using a fixed color scheme rather than dynamic colors.

## Theme Setup

The main theme composable is defined in `/home/daytona/workspace/source/app/src/main/java/com/example/huellitas/ui/theme/Theme.kt`:

```kotlin
@Composable
fun HuellitasTheme(
    darkTheme: Boolean = isSystemInDarkTheme(),
    content: @Composable () -> Unit
) {
    val colorScheme = if (darkTheme) DarkColorScheme else LightColorScheme

    MaterialTheme(
        colorScheme = colorScheme,
        typography = HuellitasTypography,
        shapes = HuellitasShapes,
        content = content
    )
}
```

### Theme Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `darkTheme` | `Boolean` | System setting | Whether to apply dark color scheme |
| `content` | `@Composable () -> Unit` | Required | Themed content |

- **Care and compassion** (primary purple)
- **Warmth and support** (soft violet)

## Shapes

The shape system uses rounded corners for a warm, friendly aesthetic:

```kotlin
val HuellitasShapes = Shapes(
    extraSmall = RoundedCornerShape(4.dp),
    small = RoundedCornerShape(8.dp),
    medium = RoundedCornerShape(16.dp),
    large = RoundedCornerShape(24.dp),
    extraLarge = RoundedCornerShape(32.dp)
)
```

### Shape Usage Guidelines

| Shape | Corner Radius | Common Uses |
|-------|---------------|-------------|
| `extraSmall` | 4dp | Small chips, indicators |
| `small` | 8dp | Buttons, small cards |
| `medium` | 16dp | Cards, dialogs |
| `large` | 24dp | Bottom sheets, large cards |
| `extraLarge` | 32dp | FABs, special containers |

## Usage Examples

### Basic Theme Application

```kotlin
import androidx.compose.material3.MaterialTheme
import com.example.huellitas.ui.theme.HuellitasTheme

@Composable
fun App() {
    HuellitasTheme {
        Surface(
            modifier = Modifier.fillMaxSize(),
            color = MaterialTheme.colorScheme.background
        ) {
            // Your app content
            MainNavigation()
        }
    }
}
```

### Accessing Theme Colors

```kotlin
import androidx.compose.material3.MaterialTheme

@Composable
fun CustomCard() {
    Card(
        colors = CardDefaults.cardColors(
            containerColor = MaterialTheme.colorScheme.primaryContainer,
            contentColor = MaterialTheme.colorScheme.onPrimaryContainer
        )
    ) {
        Text(
            text = "Custom Card",
            style = MaterialTheme.typography.titleMedium
        )
    }
}
```

### Using Accent Colors

```kotlin
import com.example.huellitas.ui.theme.DogChip
import com.example.huellitas.ui.theme.CatChip
import com.example.huellitas.model.TipoAnimal

@Composable
fun AnimalTypeBadge(tipo: TipoAnimal) {
    val badgeColor = when (tipo) {
        TipoAnimal.PERRO -> DogChip
        TipoAnimal.GATO -> CatChip
        TipoAnimal.OTRO -> GradientStart
    }

    Surface(
        shape = RoundedCornerShape(20.dp),
        color = badgeColor.copy(alpha = 0.9f)
    ) {
        Text(
            text = tipo.etiqueta,
            style = MaterialTheme.typography.labelMedium,
            color = Color.White
        )
    }
}
```
