  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Huellitas in minutes. Follow these steps to set up your development environment and start helping street animals.
  </p>

  <Steps>
    <Step title="Clone the repository">
      Clone the Huellitas repository to your local machine and open it in Android Studio.

      ```bash
      git clone https://github.com/Alfredogc21/huellitas.git
      cd huellitas
      ```
    </Step>

    <Step title="Configure the backend">
      Set up the backend URL in `app/build.gradle.kts`. For local development, the debug build automatically points to `http://10.0.2.2/huellitas/` (XAMPP local server via Android emulator).

      ```kotlin
      buildTypes {
          debug {
              buildConfigField("String", "BASE_URL", "\"http://10.0.2.2/huellitas/\"")
          }
          release {
              buildConfigField("String", "BASE_URL", "\"https://your-domain.com/huellitas/\"")
          }
      }
      ```
    </Step>

    <Step title="Sync and build">
      Open the project in Android Studio, sync Gradle dependencies, and run the app on an emulator or physical device.

      <Note>
        Minimum SDK 24 (Android 7.0) is required. Target SDK is 36.
      </Note>
    </Step>

    <Step title="Start registering animals">
      Once the app is running, tap the **Register** button to capture photos of street animals, add their location and contact details, and help them find rescue.
    </Step>
  </Steps>
</div>
