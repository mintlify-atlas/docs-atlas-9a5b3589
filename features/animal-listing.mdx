## Filtering System


**Available Filters**:
- **Todos**: Show all animals (default)
- **Perro**: Dogs only (ID: 1)
- **Gato**: Cats only (ID: 2) - Currently hidden
- **Otro**: Other animals (ID: 3) - Currently hidden

When a type is selected, the ViewModel refetches from the API:


### Sorting Options

Sort controls are provided via `FilaChipsFiltro` component:

```kotlin
FilaChipsFiltro(
    filtroActual = filtroActual,
    fechaSeleccionada = fechaSeleccionada,
    alSeleccionarFiltro = { opcion ->
        filtroActual = opcion
        if (opcion != OpcionFiltro.POR_FECHA) fechaSeleccionada = null
        scope.launch { listState.scrollToItem(0) }
    },
    alSolicitarCalendario = { mostrarCalendario = true }
)
```

**Sort Options** (defined in `OpcionFiltro`):
- **RECIENTES**: Most recently registered first (default)
- **ANTIGUOS**: Oldest registrations first
- **POR_FECHA**: Filter by specific calendar date

#### Sorting Implementation

Client-side sorting is applied after fetching:

```kotlin
private fun cuando(
    filtro: OpcionFiltro,
    fechaMs: Long?,
    lista: List<Animal>
): List<Animal> = when (filtro) {
    OpcionFiltro.RECIENTES -> lista.sortedByDescending { it.fechaRegistro }
    OpcionFiltro.ANTIGUOS -> lista.sortedBy { it.fechaRegistro }
    OpcionFiltro.POR_FECHA -> {
        if (fechaMs != null) {
            val calendarioSeleccionado = Calendar.getInstance(TimeZone.getTimeZone("UTC"))
                .apply { timeInMillis = fechaMs }
            lista.filter { animal ->
                val c = Calendar.getInstance().apply { time = animal.fechaRegistro }
                c.get(Calendar.YEAR) == calendarioSeleccionado.get(Calendar.YEAR) &&
                c.get(Calendar.DAY_OF_YEAR) == calendarioSeleccionado.get(Calendar.DAY_OF_YEAR)
            }
        } else lista.sortedByDescending { it.fechaRegistro }
    }
}
```

## Pagination

### Infinite Scroll

The app uses infinite scroll to load more animals as users reach the bottom:

```kotlin
val debeSolicitarMas by remember {
    derivedStateOf {
        val totalItems = listState.layoutInfo.totalItemsCount
        val ultimoVisible = listState.layoutInfo.visibleItemsInfo.lastOrNull()?.index ?: 0
        totalItems > 0 && ultimoVisible >= totalItems - 3
    }
}

LaunchedEffect(debeSolicitarMas) {
    if (debeSolicitarMas) {
        viewModel.cargarMas()
    }
}
```

**How it works**:
1. Monitors the last visible item in the `LazyColumn`
2. When user scrolls within 3 items of the end, triggers `cargarMas()`
3. ViewModel fetches the next page and appends results

### ViewModel Pagination Logic

```kotlin
fun cargarMas() {
    if (_isCargandoMas.value || !hayMasPaginas) return

    viewModelScope.launch {
        _isCargandoMas.value = true
        val siguiente = paginaActual + 1

        when (val resultado = fetchPagina(siguiente)) {
            is Resultado.Exito -> {
                paginaActual = siguiente
                animalesAcumulados.addAll(resultado.datos)
                hayMasPaginas = resultado.datos.size >= limitePorPagina
                _estado.value = EstadoListaAnimales.Exito(animalesAcumulados.toList())
            }
            is Resultado.Error -> { /* Silently fail */ }
        }
        _isCargandoMas.value = false
    }
}
```

- **Page size**: 10 animals per page (`limitePorPagina`)
- **Accumulation**: New results are appended to existing list
- **Stop condition**: When a page returns fewer than 10 items

## Pull-to-Refresh

Material 3 `PullToRefreshBox` enables manual refresh:

```kotlin
PullToRefreshBox(
    isRefreshing = isRefreshing,
    onRefresh = { viewModel.refrescar() },
    modifier = Modifier.fillMaxSize()
) {
    LazyColumn { /* animal cards */ }
}
```

**Refresh behavior**:
```kotlin
fun refrescar() {
    viewModelScope.launch {
        _isRefreshing.value = true
        paginaActual = 1
        hayMasPaginas = true

        when (val resultado = fetchPagina(1)) {
            is Resultado.Exito -> {
                animalesAcumulados.clear()
                animalesAcumulados.addAll(resultado.datos)
                hayMasPaginas = resultado.datos.size >= limitePorPagina
                _estado.value = EstadoListaAnimales.Exito(animalesAcumulados.toList())
            }
            is Resultado.Error -> _estado.value = EstadoListaAnimales.Error(resultado.mensaje)
        }
        _isRefreshing.value = false
    }
}
```

Resets pagination and reloads from page 1 without hiding current content.

## State Management

**Loading States**:
```kotlin
sealed class EstadoListaAnimales {
    data object Cargando : EstadoListaAnimales()
    data class Exito(val animales: List<Animal>) : EstadoListaAnimales()
    data class Error(val mensaje: String) : EstadoListaAnimales()
}
```

**State Observation**:
```kotlin
when (estadoActual) {
    is EstadoListaAnimales.Cargando -> { /* Show spinner */ }
    is EstadoListaAnimales.Error -> { /* Show error UI */ }
    is EstadoListaAnimales.Exito -> { /* Show list with filters */ }
}
```
